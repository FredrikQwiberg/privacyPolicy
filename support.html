Valu8 App Support

Last updated: February 18, 2026

Need help with the Valu8 iOS app? Contact us using the details below.

Contact Support

  • Support email: support@valu8group.com

Support Hours

  • Monday - Friday: 09:00 - 17:00
  • Time zone: CET

Expected Response Time

Initial response target: 2 business days

What to Include in Your Request

  • App version (TestFlight build number if available)
  • iPhone/iPad model and iOS version
  • Steps to reproduce the issue
  • Screenshots/video (if possible)
  • Account email/username used for testing (if relevant)

Common Issues

1. Login fails

  • Verify you are using the provided demo/test credentials.
  • Ensure device internet connection is active.
  • If issue persists, send timestamp and screenshot to support.

2. Search returns no results

  • Confirm spelling and try broader terms.
  • Try multiple company names.
  • If backend is temporarily unavailable, retry after a few minutes.

3. App appears stuck/loading

  • Force-close and reopen the app.
  • Check network connectivity.
  • Reinstall from TestFlight if needed.

Beta Testing Notes

  • This build is intended for external TestFlight testing.
  • Features and data may evolve between builds.
  • Report issues through the support email above.
